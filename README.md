# Civic-Connect

Civic-Connect is a platform designed to bridge the gap between citizens and local authorities, allowing for efficient reporting and tracking of civic issues.

## Project Structure

- **frontend/**: React + Vite application for the user interface.
- **Civic-Connect/**: Spring Boot application for the backend API.

## Getting Started

### Backend (Spring Boot)

1. Navigate to the `Civic-Connect` directory.
2. Ensure you have Java and Maven installed.
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```

### Frontend (React/Vite)

1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Technologies Used

- **Frontend**: React, Vite, Axios, Tailwind CSS (if applicable)
- **Backend**: Java, Spring Boot, Maven, PostgreSQL/MySQL (if applicable)
